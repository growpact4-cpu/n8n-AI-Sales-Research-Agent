# AI Sales Research Agent

An n8n-based AI Sales Research Agent that automates company research, identifies target customers, uncovers pain points, and generates structured sales insights for personalized outreach.

## Features

- Company research automation
- Target customer identification
- Pain point extraction
- AI-powered sales insights
- Structured output generation

## Tech Stack

- n8n
- OpenAI
- HTTP Requests
- JSON Processing

## Workflow

1. Input company information
2. Gather research data
3. Analyze target customers
4. Extract pain points
5. Generate sales insights
6. Return structured results

## Installation

1. Download the workflow JSON file.
2. Import it into n8n.
3. Configure required credentials.
4. Run the workflow.
